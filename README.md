## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the pcr-money
   ```sh
   git clone https://github.com/vieiraroger/pcr-money.git
   ```
2. Install PIP requirements
   ```sh
   pip install -r requirements.txt
   ```
